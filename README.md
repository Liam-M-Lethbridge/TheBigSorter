# the-big-sorter
The big sorter is just a tool I wanted to create to help store literally any form of ranked object information. For instance, you can store a bunch of movies with criteria like "plot, characters, visuals" and store your own personal ranking of the movies you've watched.

It's like LetterBoxd but more generalised and also more descriptive.

## Type Support for `.vue` Imports in TS

TypeScript cannot handle type information for `.vue` imports by default, so we replace the `tsc` CLI with `vue-tsc` for type checking. In editors, we need [Volar](https://marketplace.visualstudio.com/items?itemName=Vue.volar) to make the TypeScript language service aware of `.vue` types.

## Project Setup

```sh
npm install
```

### Run server for database

```sh
docker compose up -d # runs the database in the docker container in background
npm run server # runs the server
```

### Compile and Hot-Reload for Development

```sh
npm run dev # run the client
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
